# Image Files Reference

This document lists all the image files that should be placed in the `images/` directory.

## Required Images

### 1. Home Page (`index.html`)
- **File:** `images/hero-banner.jpg`
- **Description:** Horizontal banner with "Milloza - Ancient Grains, Modern Nutrition" text and millet grains in wooden bowls
- **Used in:** Hero section of home page
- **Line:** 46

### 2. About Page (`about.html`)
- **File:** `images/about-banner.jpg`
- **Description:** Banner with "Rooted in Tradition, Backed by Science" - Indian woman holding millet stalks in a golden field
- **Used in:** Hero section of about page
- **Line:** 37

### 3. Millets 101 Page (`millets-101.html`)
- **File:** `images/millets-101-banner.jpg`
- **Description:** Banner with "Millets 101: Discover the Power of Ancient Grains" and various millet grains in wooden bowls
- **Used in:** Hero section of millets-101 page
- **Line:** 37

## Image Directory Structure

```
Milloza/
├── images/
│   ├── hero-banner.jpg
│   ├── about-banner.jpg
│   └── millets-101-banner.jpg
├── index.html
├── about.html
├── millets-101.html
└── ...
```

## Notes

- All images should be in JPG format
- Images are referenced with relative paths: `images/filename.jpg`
- Make sure the `images/` directory exists (it has been created)
- Image dimensions should be optimized for web (recommended: 1200-1600px width for banners)
